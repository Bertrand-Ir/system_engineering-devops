# 0x02. Shell, I/O Redirections and Filters

This project covers input/output redirection, filters (such as `head`, `tail`, `sort`, `uniq`), and text manipulation in the Linux shell.

## Learning Objectives
- Understanding standard input, standard output, and standard error redirection
- Working with commands like `cat`, `head`, `tail`, `sort`, `uniq`, `grep`, `tr`
- Using pipes to chain commands together

## Requirements
- All scripts are written in Bash
- All scripts start with `#!/bin/bash`
- All scripts are executable
