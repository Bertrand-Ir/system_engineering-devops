# 0x03. Shell, variables and expansions

This project covers environment and local variables, shell initialization files, aliases, and arithmetic expansions in the Linux shell.

## Learning Objectives
- Understanding local and environment variables
- Managing shell initialization files (`.bashrc`, `.bash_profile`, etc.)
- Working with aliases
- Using shell expansions and arithmetic operations

## Requirements
- All scripts are written in Bash
- All scripts start with `#!/bin/bash`
- All scripts must be exactly two lines long
- All scripts must end with a new line
