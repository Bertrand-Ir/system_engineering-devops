# 0x04. Loops, conditions and parsing

This project covers Bash control flow structures, including loops (`for`, `while`, `until`), conditional statements (`if`, `elif`, `else`, `case`), and file/input parsing.

## Learning Objectives
- Understanding loops in Bash (`for`, `while`, `until`)
- Implementing conditional statements (`if`, `elif`, `else`, `case`)
- Utilizing file operators to check properties
- Performing basic parsing and string manipulation

## Requirements
- All scripts start with `#!/usr/bin/env bash`
- Line 2 of all scripts must be a comment explaining the script's behavior
- All scripts must end with a newline and be executable
