# 0x01. Shell, permissions

This project covers Linux file permissions, ownership, and user management. Each script demonstrates a specific permission-related command or concept.

## Learning Objectives

- Understand Linux file permissions and ownership
- Switch between users
- Change file permissions and ownership
- Work with user roles and groups

## Requirements

- All scripts are written in Bash
- All scripts start with `#!/bin/bash`
- All scripts are executable
