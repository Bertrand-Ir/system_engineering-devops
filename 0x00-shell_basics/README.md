# 0x00. Shell, basics

This project covers the fundamentals of navigating and working with the Linux shell. Each script demonstrates a specific shell command or concept.

## Learning Objectives

- Understand what the shell is
- Navigate the filesystem using the command line
- Manipulate files and directories
- Work with commands, built-ins, and aliases

## Files

| File | Description |
|------|-------------|
| `0-current_working_directory` | Prints the absolute path of the current working directory |

## Requirements

- All scripts are written in Bash
- All scripts start with `#!/bin/bash`
- All scripts are executable
