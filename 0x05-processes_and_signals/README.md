# 0x05. Processes and signals

This project covers Linux process management, Process IDs (PIDs), background services, signals (like `SIGTERM`, `SIGKILL`), and signal trapping.

## Learning Objectives
- Understanding processes and signals in Linux
- Managing process lifecycles (PIDs, parent-child relations)
- Sending signals to control process behavior (`kill`, `pkill`)
- Trapping and handling signals in scripts (`trap`)

## Requirements
- All scripts start with `#!/usr/bin/env bash`
- Line 2 of all scripts must be a comment explaining the script's behavior
- All scripts must end with a newline and be executable
